# VRChatキャスト募集掲示板 イラスト・画像素材作成

## イラスト・画像素材概要

VRChatキャスト募集掲示板で使用するイラストレーション、画像素材、グラフィック要素の設計仕様を定義します。ブランドアイデンティティに一致し、VRChatコミュニティに親しみやすい素材を作成します。

## イラストレーションスタイル

### 基本スタイル定義

#### アートスタイル
```
スタイル: フラットデザイン + 軽微なグラデーション
線画: なし（塗りのみ）
影: ドロップシャドウ最小限
質感: マット仕上げ
複雑さ: シンプル・ミニマル
親しみやすさ: 重視
```

#### カラーパレット（イラスト用）
```
Primary Colors:
- メインブルー: #3B82F6
- アクセントパープル: #A855F7
- サポートティール: #14B8A6

Secondary Colors:
- ウォームピンク: #F472B6
- サンセットオレンジ: #FB923C
- フレッシュグリーン: #34D399

Neutral Colors:
- ソフトグレー: #F3F4F6
- チャコール: #374151
- ピュアホワイト: #FFFFFF
```

#### 形状・構成要素
```
基本形状: 円形、角丸長方形中心
角の丸み: 8px-16px
グラデーション: 軽微な線形グラデーション（10-20%透明度差）
要素配置: 左右対称または黄金比
空間: 十分な余白を確保
```

## 画像素材仕様

### ファイル形式・サイズ

#### Web用画像
```
形式: WebP（メイン）, PNG（フォールバック）
圧縮: 85%品質
最適化: 必須

サイズ展開:
- @1x: 基準サイズ
- @2x: Retina対応（2倍サイズ）
- @3x: 高解像度対応（3倍サイズ）
```

#### イラスト素材
```
原版: SVG（ベクター）
Web出力: PNG, WebP
サイズ: 用途に応じた複数サイズ

命名規則:
illustration-[category]-[name]-[size].webp
例: illustration-hero-community-1200x600.webp
```

### レスポンシブ対応

#### ブレークポイント別最適化
```
Desktop (1200px+):
- 高解像度版使用
- 詳細なイラスト

Tablet (768px-1199px):
- 中解像度版
- 要素の適度な簡略化

Mobile (767px以下):
- 軽量版
- 重要要素のみ強調
```

## 実装ガイドライン

### CSS実装
```css
/* レスポンシブ画像 */
.illustration {
  width: 100%;
  height: auto;
  max-width: 100%;
}

/* 遅延読み込み */
.illustration[data-lazy] {
  opacity: 0;
  transition: opacity 0.3s;
}

.illustration[data-lazy].loaded {
  opacity: 1;
}

/* 装飾要素アニメーション */
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}

.decoration-float {
  animation: float 3s ease-in-out infinite;
}
```

### HTML実装例
```html
<!-- レスポンシブ画像 -->
<picture>
  <source media="(min-width: 1200px)" 
          srcset="illustration-hero-desktop-1200x600.webp">
  <source media="(min-width: 768px)" 
          srcset="illustration-hero-tablet-800x400.webp">
  <img src="illustration-hero-mobile-400x200.webp" 
       alt="VRChatコミュニティでの出会いと協働"
       class="illustration">
</picture>

<!-- SVGアイコンイラスト -->
<div class="category-icon">
  <svg class="icon icon--category" viewBox="0 0 120 120">
    <!-- SVG content -->
  </svg>
</div>
```

### パフォーマンス最適化
```
圧縮率: WebP 85%, PNG 80%
遅延読み込み: Intersection Observer API使用
プリロード: 重要な画像のみ
CDN: 画像配信最適化
キャッシュ: 長期キャッシュ設定（1年）
```

## 素材管理

### ファイル構成
```
assets/
├── illustrations/
│   ├── hero/                # ヒーローイラスト
│   ├── features/            # 機能説明イラスト  
│   ├── states/              # 状態イラスト
│   ├── categories/          # カテゴリアイコン
│   └── decorations/         # 装飾要素
├── patterns/                # 背景パターン
├── gradients/               # グラデーション素材
└── source/                  # 元ファイル（AI, Sketch等）
```

### バージョン管理
```
命名: illustration-name-v1.0.svg
更新: マイナーバージョンアップ
記録: 変更ログ管理
バックアップ: 元ファイル保持
```
