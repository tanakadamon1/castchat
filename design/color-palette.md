# VRChatキャスト募集掲示板 カラーパレット定義

## カラーパレット概要

VRChatの世界観に合わせた、信頼性と創造性を表現するカラーパレットを定義します。
デジタル空間の幻想的な美しさと、実用的なWebアプリケーションの使いやすさを両立します。

## カラー使用ガイドライン

### アクセシビリティ要件
```
・テキストのコントラスト比
  - 通常テキスト: 4.5:1 以上
  - 大きなテキスト: 3:1 以上
  - UIコンポーネント: 3:1 以上

・推奨コンビネーション
  - Gray 900 on White: 21:1 ✓
  - Gray 700 on White: 12.6:1 ✓
  - Gray 600 on White: 7.6:1 ✓
  - Primary 500 on White: 8.6:1 ✓
  - White on Primary 500: 8.6:1 ✓
```

### 使用パターン

#### 背景色
```
・メイン背景: White
・セクション背景: Gray 50
・カード背景: White
・アクティブ状態: Primary 50
・ホバー状態: Gray 100
```

#### テキスト色
```
・プライマリテキスト: Gray 900
・セカンダリテキスト: Gray 700
・プレースホルダー: Gray 500
・リンク: Primary 500
・エラーテキスト: Error 500
```

#### ボーダー色
```
・基本ボーダー: Gray 300
・フォーカスボーダー: Primary 500
・エラーボーダー: Error 500
・成功ボーダー: Success 500
```
